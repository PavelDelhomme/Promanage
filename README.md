# ProManage

ProManage est un projet de gestion de projets, incluant un backend, un frontend web et une application mobile.

## Structure du projet
- promanage-backend: Backend du projet utilisant FastAPI.
- promanage-frontend: Frontend web du projet utilisant React avec TypeScript.
- ProManageMobile: Application mobile utilisant Expo (React Native).

## Installation
1. Clonez le repository.
2. Naviguez dans le dossier du projet.
3. Utilisez Docker pour démarrer les services backend et frontend.
